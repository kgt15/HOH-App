# HoH
It is a women's safety app.It keeps track of the user's location. When the user dials the emergency code, HoH sends an SMS with the current location as google maps link to the emergency contacts. HoH also sends a push notification via GCM to emergency contacts. The differentiating feature of HoH is that it also sends a GCM to other HoH users in a 200m radius, this helps the people nearby to help the user in distress.
